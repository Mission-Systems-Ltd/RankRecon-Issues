# RankRecon Issues

This repository is the issue tracker for [RankRecon](https://www.missionsystems.co.uk/rankrecon/), a free browser extension for SERP analysis and competitor research.

**This repository does not contain source code.** It exists solely for reporting bugs, requesting features, and tracking known issues.

## Reporting an Issue

Before opening an issue:

1. Check the [documentation](https://www.missionsystems.co.uk/rankrecon/docs/) — your question may already be answered
2. Search [existing issues](https://github.com/MissionSystems/rankrecon-issues/issues) to avoid duplicates

When reporting a bug, include:

- Browser and version (e.g., Chrome 120, Firefox 121)
- RankRecon version (shown in the extension popup footer)
- Steps to reproduce the problem
- What you expected to happen
- What actually happened
- The URL of the Google search page (if relevant)

For feature requests, describe the use case — what are you trying to accomplish?

## What This Tracker Covers

- Bugs in the RankRecon extension
- Feature requests and suggestions
- Questions about functionality

## What This Tracker Does Not Cover

- General SEO questions
- Issues with websites you're analysing (bot protection, page errors, etc.)
